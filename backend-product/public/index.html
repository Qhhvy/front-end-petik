API Documentation - Products

1. Get All Products

Method: GET

Endpoint: https://api-product.sukmax.my.id/products

Description: Returns a list of all products.

          GET /products

2. Get Product by ID

Method: GET

Endpoint: https://api-product.sukmax.my.id/products/:id

Description: Returns a specific product by its ID.

          GET /products/:id

3. Create a Product

Method: POST

Endpoint: https://api-product.sukmax.my.id/products

Description: Creates a new product.

          POST /products

4. Update a Product

Method: PUT

Endpoint: https://api-product.sukmax.my.id/products/:id

Description: Updates an existing product by its ID. 

          PUT /products/:id

5. Delete a Product

Method: DELETE

Endpoint: https://api-product.sukmax.my.id/products/:id

Description: Deletes an existing product by its ID. 

          DELETE /products/:id
